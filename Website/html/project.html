Developer Appointmet System using java

This is the GUI appoiment system sytem which can be used to appoint developers to particular platform.This system is designed to appoint developer's in two platfrom either in senior platfrom or in junior platfrom.And we can also details of added developer in this system.This sstem open in main menu which has option to choose either senior or junior developer.


In senior platform we can add developer for senior platfrom and can appoint the developer in that platform.After each developer is added a develoepr number is assigned for developer which is equals to its data stored index in arraylist and can't appoint another developer in that developer number value. We can display details of added develoepr.In this system we can also terminate contract of developer and once terminated details of developer is removed.


Similar to senior developer we can add developer for junior platform in this system and appoint them and number is assigned for developers in which details of developer's are stored.Details can displayed for added develelopers.System is designed in such a way that developer number assigned for senior and developer can't be same which provide each added developer unique developer number.
